Some of the Basic Git Commands

don't put password, api keys, utillitiess files, setting files

git init
git add filename .(to add all)
git commit -m Message
git log (to see commits)
git status (to see status)
git diff (to see diff between changes)
git checkout (to revert changes)
git remote add origin link(.git) (to link to repository created on github) (origin can be anything)
git push -u origin master (to push data to master)
git rm --cached -r (to remove all files from staging area)
git branch branchname (to add branch)
git checkout branch (to change branch)
git branch (to see branches)
